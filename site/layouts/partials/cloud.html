Moon Cloud

Don't have your own Kubernetes cluster but still want to use Moon? This is now possible with Moon Cloud solution. How it works:

  1. You choose one of supported cloud platforms: Amazon Web Services, Google Cloud or Microsoft Azure;
  2. We deploy and maintain a dedicated Moon cluster for you;
  3. You pay the regular cloud billing + 25%. No need to purchase Moon license.

This approach gives you a lot of flexibility:

  • Unlimited. Run as many parallel sessions as needed at any moment of time;
  • Cost efficient. Per-minute billing and auto-scaling allow to dramatically decrease cluster cost;
  • Secure. We deliver a dedicated cluster that can optionally run in your virtual network and save data to S3 bucket you fully control;
  • No ugly tunnels. Use all regular cloud network connectivity features to connect Moon cluster to your corporate network.
